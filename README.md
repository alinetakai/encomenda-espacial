
# Encomenda Espacial da NAVE 404 🚀

📌 **Explicação:**

Projeto autoral, voltado para fins educacionais.

Trabalha o conceito de botões de envio de formulários HTML, simulando um mini sistema de “encomenda interestelar”

## Aprendizados

Botões de Envio - Formulários HTML.

🎯 **Objetivo:** 👩‍💻

Usar os botões submit, reset e button em um contexto criativo e divertido.

💡 **Recursos explorados:**

- Formulário com campos input
- Botões com type="submit", type="button" e type="reset"
- JavaScript básico com onclick
- Estilo visual moderno com CSS
## Funcionalidades

👩‍💻 **Objetivo:**

- Aprender a usar os botões submit, reset e button em um contexto criativo e divertido.

✨ **Descrição:**

- Mini sistema de “encomenda interestelar”. 
- A pessoa pode preencher o nome do planeta, o tipo de encomenda (ex: robô, combustível, pizza cósmica 😄), enviar o pedido, ver uma dica secreta com button type="button", ou resetar tudo com reset.



## Stack utilizada

- HTML
- CSS
- JavaScript


## Deploy
